Article_Classifier
==================

Little web app that classifies your text based on a corpus of New York Times articles.


###To deploy on yhat    
   	export YHAT_USERNAME="YOUR USERNAME"
    export YHAT_APIKEY="YOUR APIKEY" 
    

###To deploy on Heroku
    heroku create
    git push heroku master
